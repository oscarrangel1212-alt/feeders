Cómo usar etiquetas NFC
Configura las etiquetas físicas para que al escanearlas abran automáticamente la información del feeder.
1
Registra el feeder
Ve a la pestaña Admin, presiona + Agregar e ingresa el ID del feeder, la máquina, el estado y una descripción de la falla si aplica.
2
Obtén la URL del feeder
Cada feeder tiene una URL única. La URL que debes programar en la etiqueta tiene este formato:
https://tu-sitio.com/feeders.html?feeder=T01F03E31
3
Programa la etiqueta NFC
Descarga NFC Tools (Android/iOS) o TagWriter by NXP. Selecciona Escribir → URI y pega la URL del feeder. Etiquetas recomendadas: NTAG213, NTAG215, NTAG216.
4
Pega la etiqueta en el feeder
Colócala en un lugar visible. Al acercar el teléfono, el navegador abrirá automáticamente esta página con el estado del feeder.
5
Actualiza el estado cuando sea necesario
Desde Admin puedes editar el estado y descripción en cualquier momento. La etiqueta NFC no necesita reprogramarse — siempre apunta a la misma URL.
Para producción

Hospeda este archivo HTML en GitHub Pages, Netlify o Vercel (todos gratuitos). Esto te da una URL fija y permanente para todas tus etiquetas NFC. Los datos se guardan en el teléfono donde se editan (localStorage). Para datos compartidos entre varios teléfonos, considera agregar Supabase como backend.
